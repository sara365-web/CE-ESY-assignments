# المخزن الدائري (Circular Buffer) – مشروع بلغة C

## وصف المشروع
هذا تطبيق لمخزن دائري (Queue دائرية) بلغة C.  
يسمح بتخزين سلسلة نصية داخل المخزن، ثم قراءتها مرة أخرى، مع معالجة حالتين مهمتين:
- **الامتلاء (Overflow):** عند محاولة الكتابة في مخزن ممتلئ، يتم إظهار رسالة خطأ ولا تتم الكتابة.
- **الفراغ (Underflow):** عند محاولة القراءة من مخزن فارغ، يتم إظهار رسالة خطأ ولا تتم القراءة.

## الميزات
- تهيئة المخزن بحجم معين (يُحدد عند الإنشاء).
- دوال للكتابة (`circ_write`) والقراءة (`circ_read`).
- التحقق من الامتلاء والفراغ باستخدام متغير `count` (عدد العناصر الموجودة حالياً).
- اختبار حالتين في `main`:
  1. حجم مخزن **أصغر** من طول النص المطلوب تخزينه → يحدث تجاوز للسعة.
  2. حجم مخزن **أكبر** من طول النص → تتم الكتابة والقراءة بنجاح كامل.
